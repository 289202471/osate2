Error-Model Annex v2

Introduction

The support of the Error-Model Annex is done by several means:index.html

  • Fault and Hazard Analysis support for producing reliability report of your architecture.
  • Instance Model Viewer support for viewing error and their propagations in the underlying architecture.
  • Reliability Block Diagram: provides metrics about the overall reliability of the system, based on a gross estimation.
  • PRISM export: export an AADL model into a PRISM model to process model-checking method using the Markov-Chain formalism. We also explain the mapping into CTMC and DTMC.

These functionalities are available in the Fault Analysis menu from OSATE, as shown in the following picture. Note that most of functions apply to the instance model. To use them, you will need to select the system instance and run the expected analysis.
